## 📌 Sobre o Projeto
Este projeto foi desenvolvido em Python para ajudar usuários a estimar o consumo mensal de energia elétrica de qualquer aparelho doméstico, com base em dados simples de uso.

## 🛠️ Linguagem Utilizada
- Python 3.x

## 🔢 Fórmula Utilizada
```
consumoMensal = (potencia * horasDia * 30) / 1000
```

## 📋 Como Usar
1. Digite o nome do aparelho (ex: Geladeira)
2. Digite a potência em watts (ex: 150)
3. Digite o tempo médio de uso diário em horas (ex: 8)
4. O programa mostrará o consumo estimado em kWh/mês

## 📊 Exemplo de Saída
```
Aparelho: Geladeira
Consumo estimado: 36.0 kWh/mês
```
